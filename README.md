## 🔧 DIY 技術筆記
### 1. 儲存空間升級
- **原因：** 原本 128G 對我們來說太小了，反應較慢。
- **方案：** 直接上 240G SSD。
- **效果：** 讀取速度明顯提升。

### 2. CVT 換油注意
- 必須在油溫冷卻至指定溫度後才能進行洩放，避免電磁閥受損。
